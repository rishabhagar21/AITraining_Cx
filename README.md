## ✏️ Customization Guide

### Edit the HTML File
Open `bi-weekly-review.html` in any text editor and modify:

- **Metrics:** Change the values in metric-card sections (line ~200)
- **Progress Items:** Update bullet-list items (line ~215)
- **Achievements:** Modify success-item content (line ~225)
- **Risks:** Update risk-item descriptions (line ~240)
- **Action Items:** Change action-item details (line ~255)

### Example: Update a Metric
```html
<div class="metric-card">
    <div class="metric-value">92%</div>  <!-- Change this value -->
    <div class="metric-label">Project Completion</div>
</div>
```
